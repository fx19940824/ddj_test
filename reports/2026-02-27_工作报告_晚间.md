## 五、用户使用指南

### 方式一：命令行版本（推荐，无需配置）

```bash
cd d:\Project\ddj_test
python cli.py
```

### 方式二：GUI校准 + 自动识别

1. 运行校准工具：
   ```bash
   python tools/calibration_gui.py
   ```

2. 加载手牌截图，用鼠标框选每张牌

3. 保存模板，并重命名为：
   - 3_spade.png, 10_heart.png, joker_small.png 等

4. 运行主程序：
   ```bash
   python run.py
   ```
