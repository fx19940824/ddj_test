## 五、用户使用指南

### 使用方式一：自动识别（推荐）

1. **准备工作**（已完成）
   - 用户已创建37个卡片模板到 `config/templates/cards/`

2. **运行主程序**
   ```bash
   cd d:\Project\ddj_test
   python run.py
   ```

3. **识别手牌**
   - 点击【识别手牌】按钮
   - 选择手牌截图文件
   - 程序自动识别并填充手牌

4. **记录游戏**
   - 点击【记录出牌】记录已出的牌
   - 点击【上家出牌】记录上家的出牌
   - 查看右侧的推荐出牌方案

### 使用方式二：手动输入

1. 运行主程序：`python run.py`
2. 点击【输入手牌】按钮
3. 在对话框中输入每张牌的数量
4. 点击确定，程序生成推荐方案

### 使用方式三：命令行版本

```bash
python cli.py
```

## 六、技术说明

### 模板匹配原理

1. **多尺度匹配**: 尝试0.8-1.2倍的缩放比例，提高匹配成功率
2. **非极大值抑制**: 检测到卡片后，标记该区域，避免重复检测
3. **位置排序**: 按x坐标从左到右排序，符合手牌排列顺序
4. **阈值调整**: 默认阈值0.5，可根据实际情况调整

### 模板文件命名规范

```
格式: {rank}_{suit}.png

示例:
- 3_diamond.png    方块3
- 10_heart.png     红桃10
- J_spade.png      黑桃J
- joker_small.png  小王
- joker_big.png    大王
```
